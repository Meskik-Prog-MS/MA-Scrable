Crosswords 4.4.110 release

This release fixes a couple of rare crashes added by the last release. That release's experimental support for side-by-side views on tablets remains the focus.

Please take this survey to help determine what gets done next in Crosswords.

New with this release

  • Add side-by-side mode for tablets. For now, it's an option tablet owners will be encouraged to enable. Later it'll always be on.
  • Improve board snapshots
  • Make alerts' content scrollable (for very small screens)
  • Move "Play sound" preference
  • Remove old ways of zooming board (since modern phones have multi-touch screens)
  • Fix invitations creating unclosable games

(The full changelog is here.)

Next up

  • Take advantage of Marshmallow's new permissions model (where the app only asks for permission, e.g. to send SMS, when it needs it.)

Please let me know (email xwords@eehouse.org) what's broken and what features you'd most like to see. And if you do get a crash, please let your device report it to Google so I can see it!
