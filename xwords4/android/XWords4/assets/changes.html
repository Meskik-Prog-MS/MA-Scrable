Crosswords 4.4.106 release

This version improves chat display and reduces excess battery usage on some devices

Please take this survey to help determine what gets done next in Crosswords.

New with this release

  • Fix excess power usage by combining all green-to-red turn-length timers into one
  • Include player names in chat display
  • Revert recent change preventing multiple checks in SMS and Bluetooth invitation dialogs

(The full changelog is here.)

Next up

  • More chat improvements, especially allowing to send and receive without closing the chat window
  • Take advantage of Marshmallow's new permissions model (where the app only asks for them when it needs them.)

Please let me know (email xwords@eehouse.org) what's broken and what features you'd most like to see. And if you do get a crash, please let your device report it to Google so I can see it!
