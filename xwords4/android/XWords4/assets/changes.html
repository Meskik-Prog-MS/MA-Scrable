Crosswords 4.4 beta 90 release

Please take this survey to help determine what gets done next in Crosswords.

New with this release

  • Enable play via Bluetooth. Try it: it's great when you're within range
  • Enable invite-via-tapping (NFC/"Andriod beaming") for Bluetooth games
  • Make tray divider wider so easier to grab and drag
  • Make new game entries collapsible to save space for those who don't use them
  • fix half-height board bug
  • fix occasional crash creating a new game immediately after a delete
