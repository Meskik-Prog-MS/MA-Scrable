Crosswords 4.4.115 release

This release uses the new permissions process introduced with Android "Marshmallow": Crosswords only asks for permissions at the moment they're required.

Please take this survey to help determine what gets done next in Crosswords.

New with this release

  • Request permissions when they're needed, e.g. ask to send SMS messages when a game is being configured to communicate using SMS
  • New two-color icon for notifications (required by the newer Android that supports just-in-time permissions)
  • Use less battery by being smarter about how often to check the relay for moves (This will mostly impact those installing from F-Droid.)
  • Stop mis-identifying tablets as data-SMS-capable (only GSM phones are)
  • Allow board "squares" to be wider than tall if it fills the screen better
  • Improvements in localization for French, Catalan, Dutch, Czech and Japanese (via Weblate)

(The full changelog is here.)

Next up

  • Continue work to support WiFi Direct (currently working sporadically)
