Crosswords 4.4.111 release

This release changes the display order of games in the main screen, placing those in which it's your turn at the top and games that have ended at the bottom. And it fixes an occasional crash in side-by-side mode.

Please take this survey to help determine what gets done next in Crosswords.

New with this release

  • Display at the top of the list games in which it's the local player's turn
  • Display at the bottom of the list games that have ended
  • Fix crash launching app in side-by-side mode

(The full changelog is here.)

Next up

  • Take advantage of Marshmallow's new permissions model (where the app only asks for permission, e.g. to send SMS, when it needs it.)
