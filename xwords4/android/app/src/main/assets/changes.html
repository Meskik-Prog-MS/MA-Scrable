CrossWords 4.4.116 release

This release adds side-by-side display for large-screen devices in the "landscape" orientation. And tweaks the capitalization of the app name.

Please take this survey to help determine what gets done next in Crosswords.

New with this release

  • Enable side-by-side display for tablets
  • Capitalize the W in CrossWords: this is not a crossword puzzle game
  • In service of above, rewrite pretty much everything dialog-related. :-)
  • Fix game timers
  • Make blank-tile-picker dialog prettier
  • Remove pick-face-up option for networked games. (It didn't work anyway.)
  • Don't open a closed group when moving games to it
  • Fix occasional crash
